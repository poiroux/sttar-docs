Installation

This chapter gives instructions on installing the software. It covers:

  • Hardware and software requirements
  • Installing a workstation license
  • Installing a network license with a license server

Licensing is handled by WIBU CodeMeter, the license server that controls the rights to use Setics Sttar products. A license can live on a USB dongle or in a software license file (a license container "attached" to the machine).
