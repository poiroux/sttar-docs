# What changed in this version

This page is **per-version**: switch releases with the navbar version picker
and this page shows the changes for the selected release.

## STTAR 2.5 (Batch Engine)

The Batch Engine runs the same design engine as Advanced Designer, so it
inherits the 2.5 engine, export and data-model changes (see the Advanced
Designer **What changed** page for the full list — e.g. the new `INFCONNAME`
export column, the `EndpointCategory` attribute, GDAL/OGR 3.8.4, and the
deterministic-filter and routing fixes).

- **Requirements** — now requires **.NET Framework 4.8** (raised from 4.6.2 /
  4.7.2) and the **Visual C++ 2022** redistributable.
- The command-line interface (modes, options and return codes) is unchanged
  from 2.4.

## STTAR 2.4 (Batch Engine)

- Initial documented baseline for the Batch Engine manual.
- Command-line interface, output files, and shared installation/appendix
  chapters documented from the 2.4 release.
