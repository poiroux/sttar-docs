# Setics STTAR Cloud

**Cloud** is the hosted offering of the Setics Sttar platform — the same
fiber-network design and optimization engine delivered as a managed service
rather than a desktop install.

:::warning Draft — scope being defined
This manual is a **placeholder**. The Cloud product surface and its
documentation scope are still being settled (dedicated Cloud manual vs. reusing
the desktop manuals), and the entitlement mapping (which license unlocks Cloud)
is not yet final. The structure below exists so the platform can host the Cloud
manual as a first-class product the moment its content is ready — the pages are
intentionally empty of feature detail until then.
:::

## Where Cloud fits

| Product | License | Delivery |
|---|---|---|
| **Advanced Designer** | `STAD` | Desktop — full interactive design & optimization. |
| **Planner** | `STPL` | Desktop — restricted operator profile. |
| **Batch Engine** | `STBE` | Desktop / server — command-line batch runs. |
| **Cloud** | *TBD* | Hosted service. |

Until the dedicated Cloud content lands, the engineering concepts, data model
and deliverables are documented in the **[Advanced Designer
manual](/advanced-designer/intro)** — the Cloud service runs the same engine.
