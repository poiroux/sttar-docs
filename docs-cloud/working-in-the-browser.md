# Working in the browser

The hosted equivalents of the desktop application tabs.

:::warning Draft — pending product decision
The browser UI is not documented yet. This page will map each desktop tab
(Project, Map, Infrastructure, Architecture, Cable System, Ducts, Network
Optimization, Results) to its hosted equivalent once the Cloud interface is
finalized.
:::

## Intended scope

- The hosted workspace layout and navigation.
- Per-tab equivalents of the desktop [User interface](/advanced-designer/user-interface/).
- Differences from the desktop experience (what is available in the browser).

The design engine is the same as the desktop product, so the modeling concepts
carry over directly from the [Advanced Designer manual](/advanced-designer/intro).
