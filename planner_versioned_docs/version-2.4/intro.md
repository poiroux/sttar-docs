# Setics STTAR Planner

**Planner** (license code `STPL`) is a focused operator profile of the Setics
Sttar desktop application. It shares the same network model, map, engineering
rules and design engine as **Advanced Designer**, but limits the editing and
optimization capabilities to a planning-and-review workflow.

:::info[Shared manual]
Planner and Advanced Designer use **the same application**, so this manual
follows the same structure and shares its chapters with the Advanced Designer
manual. Where a capability is restricted under an `STPL` license, the
application disables or hides the corresponding command — the surrounding
documentation still applies.
:::

## What you can do with Planner

- **Open and review projects** — load a project, browse the map, the network
  model and the engineering deliverables.
- **Work with the geography and model** — load background maps, inspect imported
  pathways, nodes, endpoints and the active network.
- **Produce deliverables** — generate the engineering, business and
  interoperability exports (GIS/shapefile, reports, diagrams).

Editing and full optimization controls available in Advanced Designer may be
restricted under a Planner license; the exact restricted set is enforced by the
application at runtime.

## The STTAR product family

| Product | License | Role |
|---|---|---|
| **Advanced Designer** | `STAD` | Full interactive desktop design & optimization. |
| **Planner** | `STPL` | A restricted operator profile of the same app (editing and optimization capabilities are limited). |
| **Batch Engine** | `STBE` | Command-line / batch processing for automated runs. |

> Your license unlocks the manual for the product you own. Use the **version
> picker** in the navbar to switch between documented STTAR releases — each
> version has its own **What changed** page.
