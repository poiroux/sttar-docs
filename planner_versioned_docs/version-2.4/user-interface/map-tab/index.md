# Map Tab

The **Map** tab is divided into several parts:

- **Layer control panel / Legend** — organise and control how layers are drawn.
- **Information bar** — status of the action in progress.
- **The map** — the geographic view of the project, where you interact with elements.
- **Properties table** — attributes of the features selected on the map.

In addition, two contextual toolbars are available:

- **[Navigation tools](./navigation-tools)** — pan and zoom.
- **[Work tools](./work-tools)** — select, create, modify and delete elements.
