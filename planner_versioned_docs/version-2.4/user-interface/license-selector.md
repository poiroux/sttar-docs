# License Selector

The License Selector shows the license in use (highlighted) and lets you choose
a license better suited to the project being processed.

| Command | Description |
|---|---|
| **Refresh** | Restarts a license-server search on the local network and displays an updated list of available licenses. |
| **Preferred license** | Marks a license (on a given server) as the one to acquire by default when the application opens. |
| **Acquire** | Requests the selected license; on success, releases the current one. |
| **Cancel** | Exits the License Selector without changes. |
