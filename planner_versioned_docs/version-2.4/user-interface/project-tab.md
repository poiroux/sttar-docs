# Project Tab

The Project tab is divided into five parts:

- **General Project Properties**
- **Regional Properties**
- **Engineering Properties**
- **Shortcuts**
- **Project Summary Data**

> 🖼️ *Screenshot to migrate: the Project tab.*

## General Project Properties

| Setting | Description |
|---|---|
| **Project name** | Name of the current project. Defaults to "Project DATE TIME". |
| **Description** | Brief description of the current project. |
| **Configuration file** | If the project is saved, the location of the project settings file. |
| **Revision** | Date of the last backup of the current project. |

## Regional Properties

| Setting | Description |
|---|---|
| **Region** | Project location, chosen via the country's ISO-3166 code. |
| **Design Area Name** | Name of the project's geographical area. |
| **Reference coordinate system** | Map coordinate system associated with the project — data and results are exported in it. Defaults to **WGS 84 (EPSG:4326)**; a drop-down offers other systems and recently used values. Setics Sttar also accepts a full name resolved from an EPSG code, a WKT string, or a PROJ.4 string entered in the field. |
| **Currency** | Currency in which project costs are expressed. |
| **Measurement System** | Length measurement system (metric by default). |

## Engineering Properties

| Setting | Description |
|---|---|
| **Application** | Up to three applications can be defined; the application lets you specify the type of fibers for the endpoints. |
| **Configure Project Equipment Database…** | Opens the database listing the technical characteristics and costs of the equipment usable in the project (see the **Equipment Database** section). |

## Shortcuts

| Shortcut | Description |
|---|---|
| **Open a map** | Opens a map to view input data and results (see the **Map Tab** section). |
| **Open Infrastructure Configuration** | Opens the infrastructure import/create/configure window (see the **Infrastructure Tab** section). |
| **Open Architecture Model** | Opens the passive-architecture import/create/configure window (see the **Architecture Tab** section). |
| **Open Project Summary Report** | Opens a detailed Excel report: summary, configuration, estimated infrastructure quantities, and project costs (see the **Cost Model** appendix). |

## Project Summary Data

Quantitative information about the project's input data and results is available
throughout the project and is updated as the project progresses.
