# User Interface

This chapter provides a detailed description of the Setics Sttar Advanced
Designer user interface. It covers the main window, the menus, and each of the
configuration and map tabs:

- **Main Application Window** (below)
- **Menus** — File, Project, Tools, Help
- **Project Tab**, **Map Tab**, **Infrastructure Tab**, **Architecture Tab**,
  **Cable System Tab**, **Ducts & Microducts Tab**, **Network Optimization
  (Routing) Tab**, **Results Tab**
- **Equipment Database**, **Naming Rules**, **Options**, **License Selector**,
  **Plugins**

## Main Application Window

The Setics Sttar Advanced Designer main window is divided into two areas:

- **Top area** — menus and settings shortcuts for working directly with the
  different tabs (see the **Menus** section).
- **Bottom area** — the different settings and map-view tabs. By default the
  **Project**, **Infrastructure** and **Architecture** tabs are available.
