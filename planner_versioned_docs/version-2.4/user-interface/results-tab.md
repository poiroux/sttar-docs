# Results Tab

The Results tab opens once network optimization starts. Calculation time depends
on the project size; the window then refreshes with the calculated
network-optimization results.

The window has two sections: the calculated results, and the display/export
links for those results.

| Command | Description |
|---|---|
| **Export…** | Opens a window to choose the project data and results to export (see the **Export Network Data** chapter). |
| **Open Export Folder** | Opens the folder where the selected export outputs are stored. |
| **Open Project Summary Report** | Opens a detailed Excel report — overall project summary, a reminder of the settings, and the quantities (see the **Cost Model** appendix). |
