# Legend

The legend, or **layer control panel**, organises and controls the rendering of layers on the map. Right-click
a layer's name to work on that layer. The available commands depend on the layer type.

## Layer Groups

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |

## Network and Infrastructure Layers

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |
| **Set label properties** | Displays the list of attributes that can be shown on top of each symbol. Choose **&lt; No Label &gt;** to remove all displayed labels. |
| **Symbology** | Opens the layer's symbol editor to configure how features are rendered. The editor is in *easy* mode by default; *advanced* mode is enabled via **Tools ▸ Options ▸ GIS & Topology ▸ Use Advanced Symbol Editor**. |
| **Export Data** | Opens a dialog to choose where to export the GIS layer. Data is exported in the project's coordinate system; the GIS format is chosen when saving. |
| **Open Support Properties** | Opens the corresponding support configuration screen (infrastructure, node and endpoint supports only). |

## Assert Connection Layers

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |
| **Set label properties** | Displays the list of attributes that can be shown on top of each symbol. Choose **&lt; No Label &gt;** to remove all displayed labels. |
| **Symbology** | Opens the layer's symbol editor (easy/advanced as above). |
| **Export Data** | Opens a dialog to export the GIS layer (project coordinate system; format chosen on save). |
| **Show deleted items** | Creates a new layer of the deleted elements from this layer. Deleted asset-connection elements can be undeleted. |

## Markups Layers

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |
| **Add Markup Layer** | Opens a dialog to search for a markup layer. |

## Context Layers

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |
| **Add Context Layer** | Opens a dialog to search for a GIS context layer (cadastral, administrative boundaries, etc.) to display. Available on contextual layer groups only. |

## Basemap

| Command or setting | Description |
|---|---|
| **Center on the map** | Adapts the view of the map to center the elements of the layer. |
| **Custom Project Basemaps** | Add project-based basemaps. |
| **Built-in Project Basemaps** | Displays the list of basemaps that can be used. |
| **Add BaseMap** | Opens a dialog to add a Web Map Tile Service. |
