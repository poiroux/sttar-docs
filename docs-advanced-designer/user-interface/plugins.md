# Plugins

Setics Sttar Advanced Designer provides a plugin architecture to extend the
software's functionality.

Plugins are installed in the **`Plugins`** subfolder of the installation folder.
By default:

- Windows 64-bit: `C:\Program Files\Setics Sttar\Plugins`
- Windows 32-bit: `C:\Program Files\Setics Sttar\Plugins`

:::note
Plugins are optional and not included by default. Setics Sttar also offers
customized plugins to meet specific customer needs.
:::
