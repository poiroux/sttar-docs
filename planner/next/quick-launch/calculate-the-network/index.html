Calculate the Network and Make Manual Adjustments

Once the project is configured, you optimize the network, refine it by hand on the map, and export the results.

Start Network Optimization​

Once the project settings are complete, the network optimization calculation can be started. To do this, simply click on the Optimize Network button.

After the button is clicked, the Results tab opens. The calculation can take a few minutes (see the progress bar at the bottom of the screen), then the window is refreshed, showing a summary of the results obtained.

In order to view the results, the Project ▸ Map menu opens the Map tab.

The Map tab is described in detail in the Map Tab section.

Make Manual Adjustments​

While working in the map, the user has the opportunity to analyse the design result and modify it in real time.

The following actions are examples of common manual adjustments that are made in the Map tab of Setics Sttar Advanced Designer:

  • Repositioning a node or endpoint.
  • Addition of a new endpoint or node.
  • Addition of a missing reusable infrastructure.
  • Deactivation of an infrastructure interconnection, or of the distribution of a node or endpoint that you do not want to use.
  • Reassigning an endpoint or node to its parent node (e.g. modification of the service areas).
  • Modification of an attribute value associated with a node or endpoint (e.g. number of terminations, fibre count, …).
  • Modification of an attribute value associated with a section of infrastructure (e.g. actual length, specific cost, …).
  • Modification of an attribute value associated with a specific infrastructure (e.g. capacity of cabling equipment, …).

The tools are described in the Work Tools section, and map usage in the Map Tab section.

Save Results​

In order to save the results obtained by Setics Sttar Advanced Designer, click on Export in the Results tab. A new window appears:

To export your results, select the data to be exported, select a folder with the Browse… button, then select a format from the Format drop-down list, and finally click on Export. The process is developed in the Export Network Data chapter.
