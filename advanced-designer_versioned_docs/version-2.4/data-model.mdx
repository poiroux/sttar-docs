# Data model reference

This page prototypes the RFC's **schema-driven reference tables**: the table
below is rendered from a JSON artifact (`/sample-datamodel.json`) that, in
production, would be **emitted by the STTAR build per version** — so the
reference always matches the code.

<Callout type="note">
The JSON is fetched at runtime from the static bundle. In production this
artifact is versioned alongside the docs so each release shows its own schema.
</Callout>
