# Advanced Designer

The **Advanced Designer** (license code `STAD`) is the interactive desktop
workspace for fiber-optic network design and optimization.

> This is **placeholder content** for the documentation-platform validation
> draft. It exists to prove the architecture (multi-manual, versioning, i18n,
> access gating, search) works end to end — not to document the product.

## What this manual covers

- Project and network model concepts
- The optimization pipeline
- The data model reference (see **Data Model**)

Use the **version picker** in the navbar to switch between documented STTAR
releases. Each version has its own **What changed** page.
