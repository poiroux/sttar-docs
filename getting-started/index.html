Getting started

Welcome to the Setics Sttar documentation. Setics Sttar is a suite of fiber-optic network design and optimization products. This page is the common entry point — it is open to any signed-in user and is not gated by product license type.

Pick your product​

ProductLicenseWhat it doesManual

PlannerSTPLDesktop, restricted operator profile.Open
Batch EngineSTBECommand-line / server batch runs.Open
CloudTBDHosted service (same engine, in the browser).Open

Four steps to your first design​

  1. Check the requirements. Confirm your workstation (or license server) meets the System requirements.
  2. Download and install. Get the installer for your version from the Downloads page and run it; the setup assistant installs the prerequisites (.NET, Visual C++, CodeMeter) if they are missing.
  3. Set up your license. Setics Sttar uses the CodeMeter licensing system — see Licensing for workstation vs. network licenses and the CodeMeter runtime.
  4. Start designing. Launch the product and follow its getting-started flow: Advanced Designer's Quick launch, the Planner walkthrough, or the Batch Engine CLI guide.
Where to go next

Each manual's sidebar follows the same flow as step 4 in full detail — prepare data → configure → optimize → validate → report and export.
